SeraphByte

local llm runner · rust · websocket · gen8

about

SeraphByte is a local LLM runner built around a Rust/Axum backend and a live web chat interface over WebSocket. No cloud, no subscriptions — just your hardware, your models, running fast.

The web UI connects directly to the engine over a persistent WebSocket, streaming tokens in real time. Hot-swap models without restarting, pin memory between sessions, and tune generation parameters on the fly.

features
⚡
Real-time token streaming WebSocket connection streams tokens live as the model generates — no waiting for the full response.
🔄
Hot model swapping Switch between GGUF models from the UI without restarting the backend.
⬡
Pinned memory Pin context across sessions so the model remembers what matters.
◎
Generation config Tune temperature, top-p, and max tokens live from the sidebar.
🤖
Seraphine integration Powers the Seraphine Discord bot — queries flow from Discord through SeraphByte and back.
